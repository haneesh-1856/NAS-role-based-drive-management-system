File Management system platform that provides users to manage files for storing, organizing, and retrieving files  and backups with strict access control and permissions. This supports multiple user roles from basic readers to system administrators, each with clearly defined previleges. The application provies real-time storage tracking, backup and restore options, and a system monitoring dashboard. The project is hosted using Cloudflare Tunnel, ensuring faster load times, better stability, and global accessibility for all users.
